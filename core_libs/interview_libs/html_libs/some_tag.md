
# **一些html元素**

------
## **常用块级元素**
- div(块级盒子)
- h1-h6(描述标题)
- p(段落)
- ul(无序列表)
- ol(有序列表)
- li(列表项)
- br(换行)
- form(表单)
- table(表格)
- hr(下划线)

------
## **常用行内元素**
- span(行内盒子)
- a(链接)
- img(图片)
- button(按钮)
- input(输入框)
- select(下拉菜单)
- textarea(文本框)

------
## **h5系列**
- header(头部块)
- section(内容块)
- footer(底部块)
- article(文章标签)
- audio(音频标签)
- video(视频标签)
