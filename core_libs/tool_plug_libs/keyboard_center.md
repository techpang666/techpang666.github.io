
# **各大帮派快捷键集中营**
>佛系收录一些不常用的快捷键  

------
## **Vscode系列**
>ctrl+shift+p(命令面板 包治百病)  

* ctrl+shift+c(弹出终端 和`ctrl+~`打开终端不一样 默认cmd 可以设置终端为git)
* ctrl+h(查找并替换 和`ctrl+d`选择相同内容不一样)
* ctrl+shift+k(删除当前行)
* ctrl+shift+v(打开预览)
